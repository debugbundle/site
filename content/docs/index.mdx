## For AI Agents

DebugBundle is designed for agent-first consumption. Every capability is available through three equal interfaces: **CLI**, **API**, and **MCP**.

- [Agent Workflows](/docs/agent-workflows) — automation patterns, webhook pipelines, probe activation
- [MCP Tools](/docs/mcp) — Model Context Protocol server for IDE-based agents
- [`llms.txt`](/llms.txt) — machine-readable discovery file for LLMs and agents
